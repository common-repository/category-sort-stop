== Description ==

This plugin enables stop the category sort of when posting an article.

== Installation ==

1. Upload `category-sort-stop` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the ‘Plugins’ menu in WordPress

= Functions =

When you activate the plug-in, stop the category sort of when posting an article.
If you want to stop the function, please disable the plug-in.
